device-radius

Apple strips the model from the iPhone UA and never exposes the screen corner radius to the web. This infers it from viewport + DPR + iOS version, returns a value in CSS px (= pt on iOS), and flags how sure it is. Use it to round drawers, toasts, and sheets to match the hardware.
